# Assets Directory

Place your static assets here:
- Images (PNG, JPG, SVG, etc.)
- Fonts (TTF, WOFF, WOFF2)
- Icons
- Other static files

Example structure:
```
assets/
├── images/
│   ├── logo.png
│   └── background.jpg
├── fonts/
│   └── CustomFont.ttf
└── icons/
    └── favicon.ico
```

These files will be copied to the build output automatically.
